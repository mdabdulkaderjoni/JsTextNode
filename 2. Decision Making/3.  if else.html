if...else statement



Sometimes you want to execute a piece of code in case of the expression in the if statement evaluates to false. You can use the 
second form of the if statement which is known as if else statement. The following illustrates the syntax of the if else statement:


  if(expression){
      statement block1;
  }
  else{
      statement block2;
  }


  


The following example demonstrates how to use the if else statement:

      let x=Number(prompt('Enter a number'));

      if(x > 0){
          console.log("\nThe number x is greater than 0");
      }
      else{
          console.log("\nThe number x is less than or equal zero");
      }

This program asks users to enter a number. It displays the corresponding message based on the input value.


Nested if....else statement
The general form of a nested if...else statement is,

  if( expression )
  {
      if( expression1 )
      {
          statement block1;
      }
      else 
      {
          statement block2;
      }
  }
  else
  {
      statement block3;
  }



if expression is false then statement-block3 will be executed, otherwise the execution continues and 
enters inside the first if to perform the check for the next if block, where if expression 1 is true 
the statement-block1 is executed otherwise statement-block2 is executed.

Example:

      let a, b, c;

      if(a > b)
      { 
          if(a > c)
          {
              console.log("a is the greatest");
          }
          else 
          {
              console.log("c is the greatest");
          }
      }
      else
      {
          if(b > c)
          {
              console.log("b is the greatest");
          }
          else
          {
              console.log("c is the greatest");
          }
      }
