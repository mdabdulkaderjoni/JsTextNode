Decision making


Decision making is about deciding the order of execution of statements based on certain conditions or repeat a group of statements 
until certain specified conditions are met. 

Javascript language handles decision-making by supporting the following statements,

1. if statement
2. switch statement
3. conditional operator statement (? : operator)
4. break & continue
5. goto statemen
                                                    
if else Statement


The if-else statement  is used to perform the operations based on some specific condition.
The operations specified in if block are executed if and only if the given condition is true.

There are the following variants of if statement

1. If statement
2. If-else statement
3. If else-if ladder
4. Nested if



Here are the relational operators, as they are known, along with examples:

1. >     greater than              5 > 4 is TRUE
2. 	<     less than                 4 < 5 is TRUE
3. >=    greater than or equal     4 >= 4 is TRUE
4. 	<=    less than or equal        3 <= 4 is TRUE
5. ==    equal to                  5 == 5 is TRUE
6. !=    not equal to              5 != 4 is TRUE
