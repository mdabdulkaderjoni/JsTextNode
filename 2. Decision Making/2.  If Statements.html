if statement



The If statement is used when we want to test a condition, which is evaluated(মূল্যায়ন)for a boolean value- true or false.

1. If this condition is true then the statement or a block of statements connected to the If statement is executed.

2. If this continue is false then the statement or a block of statements connected to the If statement is not executed.
   structure of an if statement is as follows:

if ( statement is TRUE ){
    Execute this line of code
}

Meaning of If Statement :

1. It Checks whether the given Expression is Boolean or not !!
2. If Expression is True Then it executes the statement otherwise jumps to next_instruction


                            
 
Sample Program Code :

      let a=5,b=6,c;
      c = a + b ;

      if(c==24)

          console.log("Execute me 1");

          console.log("Execute me 2");


  Output : Execute me 2

Code will be executed if condition statement is True.
Non-Zero Number Inside if  means “TRUE Condition”

    let a=1, b=-10, c=0;
	
    if(a==10){
        console.log("First condition is true\n");
    }
	
    if(b){
        console.log("Second condition is true\n");
    }
	
    if(c){
        console.log("Third condition is true\n");
    }
	

    Output: First condition is true 
            Second condition is true 

In this program there are three conditions

1. if(a==10)
   This test condition is true because the value of a is 10.

2. if(b)
   This test condition is true because the value of b is -10 that is a non zero value.

3. if(c)
   This test condition is not true because the value of c is 0.

Example of multiple if statements

We can use multiple if statements to check more than one conditions.

      let x, y;
      x=Number(prompt('Enter a number'))
      y=Number(prompt('Enter a number'))

      if (x>y){
          console.log("x is greater than y\n");
      }

      if (x<y){
          console.log("x is less than y\n");
      }

      if (x==y){
          console.log("x is equal to y\n");
      }

      console.log("End of Program");

In the above example the output depends on the user input.
